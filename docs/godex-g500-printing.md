# GoDEX G500 Printing

Documento historico consolidado.

La configuracion vigente esta en:

- `docs/print-service.md`
- `docs/windows-bridge-install.md`
- `docs/godex-g500-checklist.md`

Estado final validado:

- `printer_key`: `kiosko_godex_g500`
- `GODEX_PRINT_TRANSPORT`: `tcp_9100`
- `GODEX_HOST`: IP de la GoDEX en la LAN
- `GODEX_PORT`: `9100`
- Etiqueta fisica: `80x50 mm`

`GODEX_PRINTER_HOST` y `GODEX_PRINTER_PORT` se mantienen como alias temporales para instalaciones antiguas.
